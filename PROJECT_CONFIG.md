# 项目配置文件

## 测试配置

### 测试股票数量
- **默认测试数量**: 50只股票
- **说明**: 每次测试时，只爬取前50只股票，避免全量数据测试耗时过长

### 测试模式
- **测试模式**: 启用
- **说明**: 在开发和调试时，使用测试模式可以快速验证功能

## 爬虫配置

### 多线程爬虫
- **最大调用次数**: 5000次
- **消费者线程数**: 5个
  - SinaWorker (新浪财经)
  - EastMoneyWorker (东方财富)
  - AkshareWorker (Akshare)
  - BaostockWorker (Baostock)
  - YahooFinanceWorker (Yahoo Finance)

### 请求间隔
- **最小间隔**: 0.5秒
- **说明**: 避免过于频繁的请求，防止被封

## 数据库配置

### 数据库路径
- **SQLite数据库**: `/Users/riching/work/hywork/db/sqlite/full_a_stock_cache.db`

### 数据库连接
- **连接模式**: WAL模式
- **同步模式**: NORMAL
- **说明**: 提高并发性能

## 数据源配置

### 可用数据源
1. **新浪财经** - API接口，支持实时和历史数据
2. **东方财富** - API接口，支持实时和历史数据
3. **Akshare** - Python库，支持实时和历史数据
4. **Baostock** - Python库，支持实时和历史数据
5. **Yahoo Finance** - Python库，支持实时和历史数据

### 数据源优先级
1. 东方财富 (稳定、快速)
2. Baostock (免费、专为A股设计)
3. Yahoo Finance (国际化、稳定)
4. Akshare (数据质量高)
5. 新浪财经 (API不稳定)
